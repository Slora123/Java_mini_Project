# Recycling Quest (JavaFX + JDBC)

## Quick Start

**For Windows users**: See [SETUP_WINDOWS.md](SETUP_WINDOWS.md) for detailed setup instructions.

**For Mac/Linux users**:
- Ensure Java 17+ and Maven installed
- Build: `mvn clean package`
- Run: `mvn javafx:run`
